# 2048-terminal
A terminal version of "2048" puzzle game written in Python using the curses library.

##Features:
- Terminal interface
- Arrow key controls
- Colored tiles
- Main menu
- Rules screen
- Victory/Defeat screen
- Random generation of new tiles

##Controles:
- → ← ↑ ↓ — move tiles
- Q — quit the game

##Requirements:
- Python 3
- Curses

On Linux/macOS the "curses" module is included by default.
On Windows additional installation may be required.
